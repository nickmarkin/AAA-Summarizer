Academic Achievement Award System — User Guide

This system tracks academic achievements for the UNMC Department of Anesthesiology. It supports:

  • Faculty self-reporting via quarterly web surveys
  • REDCap import for legacy data
  • Departmental data entry for awards and evaluations
  • Division chief review and verification
  • Report generation for AVC documentation

A. Create Academic Year
  1. Go to Configuration → Academic Years
  2. The system auto-creates the current year, but you can add future years
  3. Make sure the correct year is selected in the top-right dropdown
B. Update Faculty Roster
  1. Go to Faculty Roster
  2. Import from Faculty Calculator CSV, or add faculty manually
  3. Set division, rank, and AVC eligibility for each faculty member
  4. Mark inactive faculty with an end date (they won't receive surveys)
C. Set Up Divisions
  1. Go to Configuration → Divisions
  2. Create divisions if needed (Cardiac, Critical Care, etc.)
  3. Assign a Division Chief to each division

A. Create a Campaign
  1. Go to Survey Campaigns
  2. Click New Campaign
  3. Select quarter (Q1-Q2, Q3, or Q4) and academic year
  4. Set open and close dates
  5. Save the campaign
B. Send Invitations
  1. Open the campaign and click Create Invitations
  2. System creates unique links for all active, survey-eligible faculty
  3. Click Send Emails to send invitation emails
  4. Faculty click the link to access their personal survey
C. Monitor Progress
  • View response status on the campaign page
  • Send reminders to those who haven't completed
  • Faculty can save drafts and return later

Some data comes from department tracking, not faculty self-reporting:

  1. Go to Departmental Data
  2. Select a faculty member
  3. Enter data for:
    • Evaluations: New innovations, MyTip winner/count
    • Teaching Awards: Top 25%, 25-65%, Teacher of Year
  4. Points are calculated automatically

A. Enable Review Mode (Admin)
  1. After Q4 survey closes, go to Dashboard
  2. Click Enable Review Mode
  3. This unlocks verification features for division chiefs
B. Division Chief Review Process
  1. Division chiefs access their portal (via bookmarked link)
  2. Click Division Dashboard link
  3. Click each faculty name to review their activities
  4. For each activity, choose:
    • Verify - Activity is correct
    • Flag - Needs correction/clarification
    • Strike - Remove (erroneous/invalid)
  5. Use Verify All for quick approval of all activities
  6. Once all faculty verified, click Verify Division
C. Track Progress
  • Dashboard shows progress bar for each division
  • Configuration → Divisions shows overall verification status

  1. Go to Reports
  2. Available exports:
    • Points Summary CSV - All faculty points in spreadsheet format
    • Faculty Summaries - Individual PDF/Markdown reports
    • Activity Reports - By activity type
  3. Select faculty and format, then download

Each faculty member has a personal portal link they can bookmark:

  • View current point totals by category
  • See all submitted activities
  • Access open surveys to submit new activities
  • Division chiefs see a link to their division dashboard

Portal links are in the format: /my/<token>/
Export all portal links from Faculty Roster → Export Portal Links

  • Academic Year selector (top-right) controls which year's data you see
  • CCC Membership is tracked on faculty profiles and carries forward
  • Annual/carry-forward items (board prep, pathways, etc.) only need to be reported once per year
  • Incomplete submissions are flagged but still included in reports
  • REDCap import can supplement or replace web survey data
  • Division verification provides audit trail for AVC documentation
