Configuration Format

The configuration JSON file should contain the following structure:

{
  "point_values": {
    "activity_type_key": 5,
    "another_activity": 10
  },
  "categories": {
    "citizenship": {
      "name": "Citizenship",
      "subsections": [...]
    }
  },
  "category_order": ["citizenship", "education", ...],
  "category_names": {
    "citizenship": "Citizenship",
    "education": "Education"
  }
}

Tip: Download the current configuration first to see the exact format and make modifications from there.
