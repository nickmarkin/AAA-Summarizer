How Year-Based Configs Work
  • Each academic year can have its own survey configuration.
  • All campaigns (Q1, Q2, Q3, Q4) within the same year share the same config.
  • Changes to a year's config affect all quarters in that year.
  • To set up a new year, use Copy Config to New Year to duplicate from a previous year or the default.
  • Years without a custom config use the default from survey_config.py.
